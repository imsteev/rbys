## Usage
```bash
rbys <n>

      n (int, optional) number of bytes to generate. default is 32.
```

### executable
```bash
bun build ./index.ts --outfile rbys --compile

# or you can run convenience script in package.json:
bun run build
```

### develop
To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
